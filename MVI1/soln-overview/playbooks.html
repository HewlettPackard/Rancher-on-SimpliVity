# Ansible playbooks

The Rancher-on-SimpliVity Ansible playbooks are available to download at https://github.com/HewlettPackard/Rancher-on-SimpliVity/.

This document shows you how to:

  • Prepare the VM templates
  • Create the Fedora Ansible host
  • Run the playbooks to create a highly available admin cluster, deploy Rancher and then create an user cluster
  • Configure storage using CSI

Deploying HPE SimpliVity hardware is specific to your environment and is not covered in this document. HPE SimpliVity documentation is available in the Hewlett Packard Enterprise Support Center.
